# Color Palettes Reference

This document provides pre-designed color palettes for different app types and moods.

## Dark Mode Palettes

### Indigo Dreams (Default)
Modern, professional, calming dark theme.

```css
:root {
  /* Backgrounds */
  --bg-primary: #0D0D1A;
  --bg-secondary: #1A1A2E;
  --bg-tertiary: #252540;

  /* Brand */
  --primary: #6366F1;
  --primary-light: #818CF8;
  --primary-dark: #4F46E5;
  --primary-gradient: linear-gradient(135deg, #6366F1, #8B5CF6);

  /* Accent */
  --accent: #8B5CF6;

  /* Semantic */
  --success: #22C55E;
  --warning: #F59E0B;
  --error: #EF4444;
  --info: #3B82F6;

  /* Text */
  --text-primary: #FFFFFF;
  --text-secondary: #94A3B8;
```

**Best for:** Productivity apps, todo lists, note-taking, project management

## Semantic Colors

Always include these for consistent feedback:

```css
/* Success States */
--success: #22C55E;
--success-bg: rgba(34, 197, 94, 0.1);
--success-border: rgba(34, 197, 94, 0.2);

/* Warning States */
--warning: #F59E0B;
--warning-bg: rgba(245, 158, 11, 0.1);
--warning-border: rgba(245, 158, 11, 0.2);
```

## Glass Effects

### Dark Mode Glass

```css
/* Standard glass */
background: rgba(255, 255, 255, 0.05);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.1);

/* Elevated glass */
background: rgba(255, 255, 255, 0.08);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.12);

/* Interactive glass (hover) */
background: rgba(255, 255, 255, 0.1);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.15);
```

## Gradients

### Common Gradients

```css
/* Primary gradient (button, FAB) */
background: linear-gradient(135deg, var(--primary), var(--accent));

/* Success gradient (progress) */
background: linear-gradient(90deg, var(--primary) 0%, var(--success) 100%);

/* Sunset gradient */
background: linear-gradient(135deg, #F59E0B, #EF4444);

/* Ocean gradient */
background: linear-gradient(135deg, #14B8A6, #3B82F6);

/* Purple gradient */
background: linear-gradient(135deg, #8B5CF6, #EC4899);

/* Dark overlay */
background: linear-gradient(to top, rgba(0,0,0,0.8) 0%, transparent 100%);
```

## Contrast Requirements

### Minimum Ratios (WCAG AA)

| Element | Minimum Ratio |
|---------|---------------|
| Normal text | 4.5:1 |
| Large text (18px+) | 3:1 |
| Bold text (14px+) | 3:1 |
| UI components | 3:1 |

### Safe Color Combinations (Dark Mode)

| Foreground | Background | Ratio |
|------------|------------|-------|
| #FFFFFF | #0D0D1A | 17.5:1 |
| #FFFFFF | #1A1A2E | 12.5:1 |
| #94A3B8 | #0D0D1A | 7.2:1 |
| #94A3B8 | #1A1A2E | 5.7:1 |
| #64748B | #0D0D1A | 4.6:1 |
| #64748B | #1A1A2E | 4.2:1 (borderline) |
| #475569 | #1A1A2E | 2.9:1 (fails) |
