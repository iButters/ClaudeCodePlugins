## Model Selection Decision Tree

```
START
│
├─ Is multimodal (images/video) primary?
│   └─ YES → Gemini Pro 3.0 or Opus 4.5
│
├─ Is the task primarily coding?
│   ├─ Agentic/complex → Sonnet 4.5
│   ├─ Code generation → Codex or Sonnet 4.5
│   └─ Simple transforms → Haiku 4.5
│
├─ Task complexity?
│   ├─ Simple (classification, extraction)
│   │   └─ Haiku 4.5
│   ├─ Medium (analysis, structured output)
│   │   └─ Sonnet 4.5 or GPT 5.1
│   └─ Complex (research, creative, multi-step)
│       └─ Opus 4.5
│
├─ Priority?
│   ├─ Speed → Haiku 4.5
│   ├─ Cost → Haiku 4.5
│   ├─ Quality → Opus 4.5
│   └─ Balance → Sonnet 4.5
│
├─ Special requirements?
│   ├─ Function calling → GPT 5.1 or Sonnet 4.5
│   ├─ JSON mode → GPT 5.1
│   ├─ Context caching → Gemini Pro 3.0
│   └─ Extended thinking → Opus 4.5
│
└─ Default recommendation: Sonnet 4.5
```

## Migration Considerations

### Claude Opus → Sonnet

| Aspect | Change Needed |
|--------|---------------|
| Quality | Slight reduction, usually acceptable |
| Speed | 3x faster |
| Cost | 5x cheaper |
| Prompts | More explicit instructions may be needed |

### Sonnet → Haiku

| Aspect | Change Needed |
|--------|---------------|
| Complexity | Reduce task complexity |
| Prompts | Simplify and shorten |
| Expectations | Lower for nuanced tasks |
| Use cases | Classification, simple transforms |

### Cross-Family Migration

| From | To | Key Changes |
|------|-----|-------------|
| Claude → GPT | Adjust system message usage, add JSON mode if needed |
| GPT → Claude | Leverage extended context, adjust for thinking style |
| Any → Gemini | Explore multimodal, use caching for repeated context |
